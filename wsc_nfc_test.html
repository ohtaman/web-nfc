WSC NFC 書き込みテスト

Wi-Fi Simple Configuration (WSC) バイナリを構築して NFC タグに書き込むテスト。 Android + Chrome で動作します。

① WSC バイナリで書き込み

実験的

application/vnd.wfa.wsc の MIME レコードとして書き込みます。Android が WSC として認識すれば Wi-Fi 自動接続されるはず。

構築されるバイナリ（hex） 
入力するとここにプレビューされます

② テキスト形式で書き込み（比較用）

WIFI:S:...;T:WPA;P:...;; を text レコードとして書き込みます。QR コード用のフォーマットなので、NFC 経由では自動接続されない可能性が高い。

③ NFC タグを読み取る

書き込んだタグの中身を確認します。レコードタイプ、MIME タイプ、データを表示します。
